#description

## erp x0.1

One mini ERP system made for final exam.

gulp + mongodb + express

# How to start

install dependencies:

```
  $ cd . && npm install
```

run the app:

```
  $ DEBUG=erp:* npm start
```

debug:

```
node-debug app.js --debug=3000

```
